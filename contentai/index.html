Comment connecter Make.com

1

Créer un compte Make.com

Rendez-vous sur make.com et créez un compte gratuit.

Le plan gratuit suffit pour commencer (1 000 opérations/mois).
2

Importer le scénario

Téléchargez notre template et importez-le dans Make.com.

Dans Make → Scenarios → Import Blueprint

3

Connecter vos réseaux sociaux

Dans le scénario importé, cliquez sur chaque module réseau social et connectez votre compte.

Module Instagram → Add connection
Module TikTok (Late) → Add connection
Module Facebook → Add connection
4

Récupérer l'URL Webhook

Cliquez sur le premier module du scénario (Webhook) → copiez l'URL générée.

https://hook.eu1.make.com/xxxxxxxx...

Collez cette URL dans l'onglet Connexion ci-dessus.

5

Activer le scénario

Dans Make.com, activez le scénario avec le bouton ON en bas à gauche. C'est tout — ContentAI s'occupe du reste.
